毕业了，上班了

发表于 2017-07-29 | 分类于 老李的杂货铺 | | 阅读次数

一转眼时间，上班1个月了都。已经记不得，这是第几次在感叹时光飞逝了。如今离开了校园，成为了社会李。30天过去了，还清晰地记得，当时一个人背着行李上了812，恰巧碰到北京史上最强暴雨，思绪也跟着雨水一同从高处往下砸落。

下班后，进去住宿的房间。打开铁门，黑乎乎的，什么也看不清。鼻子倒是比眼睛先感受到点什么：久久没有打扫过的阴霉味和臭袜子味搅和在一起的奇怪味道。摸到左手旁的开关，一合一闭，一开一关，反复好几遍都没有反应。从同屋的人那得知，它的正常工作方式就是这样。后来发现次卧的灯跟门口的灯一个德行。无所谓啦，就这样吧。没有了期望，也就觉得没什么了。刚踏出校门半步，就被现实狠狠地打了一大嘴巴子。

安定下来后，拨通了爸妈的电话，跟他们分享了北漂的第一个窝。住了一晚上，第二天一醒来，一个人不知道要在房间里干些什么，于是逃回熟悉的学校。这时留在学校的同学也没几个了。去楼下的超市，买来了啤酒，花生蚕豆。每次聚会只意思一下喝上一小口的我，几瓶酒很快就下肚。嘴巴不停的说，说过去逃课时光，说之后有什么计划安排。不敢停下来，不愿意去面对目前自己的“狗样”。首个周末就这样熬过来了。

现在是一名真正的程序员了。 一波三折进入现在的公司，别人眼中的高富帅行业，互联网金融。它也将是我简历上的第一份工作经历。高档写字楼，舒适的办公环境，每个工位上的双显屏和人体工学椅，周围的一切都很互联网。进门右转的IPO会议室开始忙碌了起来。上班累了偶尔意淫下：上市后，拿着期权，迎娶白富美，也是可以的。

开始上班后，业务上各种不适应，一大推金融交易相关的专业名词。比如什么持仓，委托，增量，这些第一次听说过的词。百度一搜，大篇大篇的专业性文章，看得晕头转向。由于公司的产品是windows桌面软件，开发环境不得不从自己熟悉的mac转向win。就连VS都得开始熟悉（仅在上C语言上机课时在win下写过代码。所在的部门属于sever端，负责编写各大券商的交易接口业务。

互联网金融算是传统软件行业吧，工作压力和强度也都没有一线BAT大厂那样的高。暂时手头上的几个需求，工作量也不大。平时上班呢，打交道最多的是运维和测试了。运维部署环境，测试提BUG。工位后面的小伙伴，前一天被老大怼其一天的代码量不到百行，第二天就被HR小姐姐领去签辞职报告了。从上班到离职就两周时间。我现在的工位，上个月被辞退的妹子坐着。亲眼目睹过这些后，只能不断地提醒自己不要掉以轻心。好好上班，尽快地熟悉业务，上手代码。留住这一席之位。

有了一定规模的大公司，每个人基本上只用负责一块内容，人尽其责，而这将会导致自己成长慢，学不到东西。虽然可以把一块做的很精，但有很多东西没有什么深度，如果长期维护将会导致停止进步。意识到这些之后，在熟悉公司工作节奏的同时，一直在寻找一种积极的姿势，以便在公司里面快速学习成长。

摸爬滚打了一个月了，悟出点经验来了。每一个小零件出问题了，整台机器可能就不能运转了。得学会一点一点地拆解问题。问问题，体现了思维过程，暴露了惰性。所以能google解决的，就不要问人。也确定了学习路径：从公司的源码出发，填补知识盲点，学大牛的设计模式和架构。以小需求为出发点，参与进大型项目中。跟进代码调试熟悉框架，从中学习，吸取养分。在大公司待着，我们只需跟随成功人的脚步，把先进和优秀的东西都装进知识库里。这就是所谓的站在巨人的肩膀上去前进吧。

上一篇博客，还是在学校码的字。那时候，我还有马妮。白天在图书馆写博客，投简历，准备找工作。5点30准时等你下班，给你电话，然后吃完玩逛。那段时间虽然有找工作的焦虑，却是一段很美好的时光。现在的我们说上一句话都变得不太可能了。知道你在那里，而我不会再去。这段时间的每一天，无论开心还是难过，还是平常得不能再平常的一天，都会想起你。写下这段文字的时候很疼，老浮现两个人在一起的画面。现实告诉我，李少杰，你要独立面对。

慢慢开始习惯一个人，慢慢认清了自己。以上，致即将逝去的青春！愿出走半生，归来仍是少年。
