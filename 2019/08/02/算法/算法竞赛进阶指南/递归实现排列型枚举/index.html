递归实现排列型枚举

发表于 2019-08-02 | 分类于 算法竞赛进阶指南 | | 阅读次数

题目

把 1~n 这 n 个整数排成一行后随机打乱顺序，输出所有可能的次序。

输入格式

输入一个整数n。

输出格式

按照从小到大的顺序输出所有方案，每行1个。
首先，同一行相邻两个数用一个空格隔开。
其次，对于两个不同的行，对应下标的数一一比较，字典序较小的排在前面。

输入样例

3

输出样例

思路

递归，二进制状态压缩，顺序枚举

关于递归

并列型

  1. 前后关系：从小到大枚举每位数是否被选
  2. 子问题状态：u和state(被选情况)
  3. 递归结果：无返回值，形参state
  4. 递归出口：剪枝，结束u == n

遍历过程中u的状态相同
