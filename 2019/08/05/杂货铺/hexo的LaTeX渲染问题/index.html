hexo的LaTeX渲染问题

发表于 2019-08-05 | 分类于 老李的杂货铺 |

尝试过配置几次hexo，网上也找了好些的解决办法，均以失败而告终。

临时的解决办法

在md中嵌入html代码，通过js使用MathJax引擎进行渲染

<script type="text/x-mathjax-config">
            MathJax.Hub.Config({
              tex2jax: {inlineMath: [['$','$'], ['\\(','\\)']]},
              showMathMenu: false
            });
</script>

<script type="text/javascript" src="https://cdn.acwing.com/static/MathJax-2.6-latest/MathJax.js?config=TeX-AMS-MML_HTMLorMML"></script>
使用Anaconda客户端安装xgboost(第三方包)
递归求解等比数列求和
